# langingPage_fisioterapia
Projetando o meu Futuro:  minha carreira na web. 

Vamos utilizar os recursos da T.I para aprimorar nossa profissão do futuro.

Você precisa desenvolver uma landing page, que irá ser "utilizada" na sua futura profissão.
Nessa landing page, crie alguns recursos que serão úteis para mostrar seu trabalho.

Um exemplo:
"Serei um nutricionista, e irei desenvolver uma página que fala sobre meu trabalho, e também fornece ao usuário, uma calculadora de IMC fácil e rápido para utilizar"

"Serei um advogado, e irei desenvolver uma página que pode dar dicas rápidas baseadas em alguns artigos da lei"

"Serei um educador físico, e irei desenvolver uma página, que fala sobre a importância de atividades físicas e uma maneira de calcular se a pessoa é sedentária ou ativa em exercícios."

Use sua criatividade para desenvolver esta landing page que pode vir a ser uma porta de entrada para sua profissão do futuro, pode utilizar também uma foto sua nesta página, podendo pedir para a I.A colocar você em um cenário da sua futura profissão.

Hospede essa landing page no github pages.

O github pages, é uma forma simples e rápida de colocar uma página html no ar. Utilize seus conhecimentos do github para isso.
